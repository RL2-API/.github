# RL2.API
This organisation was made to create and maintain custom modding tools for Rogue Legacy 2.

# Members
- TacoConKvass

# Tool list
| Tool | Description |
|--|--|
| [RL2.ModLoader](https://github.com/RL2-API/RL2.ModLoader) | Custom mod loader, not reliant on BepIn. |
| [RL2.API](https://github.com/RL2-API/RL2.API) | Modding API for altering and addding content to the game. |
| [RL2.ModLoader.Website](https://rl2-modloader.pages.dev) | A website about the mod loader. Also has a [mod hub](https://rl2-modloader.pages.dev/mods). |
| [RL2.ModLoader.Website.DatabaseQuery](https://rl2-modloader-db.onrender.com) | An API endpoint for querying the database storing data about mods on the [mod hub](https://rl2-modloader.pages.dev/mods). |
